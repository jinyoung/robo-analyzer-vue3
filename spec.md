지금 UML을요 NEO4j데이터를 기반으로 하되 vueflow를 통해서 그리도록 할 수 있을까요?

즉 UML 관련 생성은 이제 백엔드로 부터 요청하는게 아니라 전달받은 neo4j 데이트를 활용해서 uml을 그리는것!, 우선은 기존로직 냅두고 vueflow방식을 새로 만들어줘 (주석으로 확실히 구분)


기존 방식인 백엔드 로직을 참고하는데, 이걸 그대로 따라하지말고 vueflow로 하면서 아주최적화하여 더 깔끔하고 안전하고 정확하게 진행 

필요한기능
1. 깊이 조절 UI 추가 , 즉 클래스다이어그램에서 클래스간에 깊이를 몇단계까지 보여줄건지? , 5면 5단계까지 보여줌
2. 특정 클래스를 클릭하면 확장할 수 있도록함 (더블클릭 or 클릭시 UI 표시) 노드 패널에서 클래스다이어그램 생성 대신, 확장으로 하던지 등등 여러 방법중 가장 친화적인걸 사용

3. neo4j 데이터를 활용해서 uML로 생성해야하는거임 

일단은.. neo4j 데이터가 어떻게 구성이 되어있는지 확인이 필요할것같은데

아래의 처리로직은 필수임 
 value object DEPENDENCY 필터링-> DEPENDENCY 타입 중 is_value_object가 true인 것은 제외  , 상위 클래스에 이미 소유 관계(COMPOSITION/AGGREGATION/ASSOCIATION)가 있으면 노이즈로 판단, 상속 체인을 거슬러 올라가 상위 클래스에 소유 관계가 있으면 하위 클래스의 DEPENDENCY 제거


즉, 클래스이릠을 검색하고 선택하는것까진 동일하나, 선택되었을때 설정된 깊이에따라서 보여주고 확장까지 가능하도록 하는거임 이해됨?
